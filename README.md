# TypeScript 重构 Axios

基于TypeScript重构Axios


## 功能点

除了axios的基本功能之外

1. 在浏览器端使用 XMLHttpRequest 对象通讯
2. 支持 Promise API
3. 支持请求和响应的拦截器
4. 支持对请求异常的捕获和处理
4. 支持请求数据和响应数据的转换
5. 支持请求的取消
6. JSON 数据的自动转换
7. 客户端防止 XSRF

## 知识点

* 掌握了TypeScript的基础知识，能运用在实际的项目中
* 了解Axios的实现原理
* 了解了Axios设计HTTP知识
